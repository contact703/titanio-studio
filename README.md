# 🧠 Tita Memory

Memória persistente da Tita — sincronizada automaticamente via GitHub.

## Arquivos
- `MEMORY.md` — memória de longo prazo (projetos, decisões, contexto)
- `contexto-ativo.md` — estado atual dos projetos
- `memory/` — notas diárias

## Como funciona
Toda sessão importante atualiza esse repo automaticamente.
