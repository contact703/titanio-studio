# iOS Specialist — Lições Aprendidas

## 🔴 Regras Críticas
- **NUNCA usar EAS** — react-native RC quebra com EAS build. Sempre `xcodebuild` local
- **prebuild DEVE terminar completamente** antes de qualquer `xcodebuild`
- **Verificar ReactCodegen** antes do archive: `find ios -name "*.mm" -path "*/ReactCodegen/*"`
- **pod install** é obrigatório após cada prebuild limpo

## 2026-03-19

### Build iOS GospIA v2

**Problema 1:** expo-av header not found
- **Causa:** versão incompatível
- **Fix:** atualizar expo-av no package.json

**Problema 2:** ReactCodegen .mm not found (BUILD FAILED)
- **Causa:** prebuild não terminou antes do `xcodebuild archive`
- **Fix:** sempre aguardar "✔ Expo prebuild is complete!" antes de continuar
- **Regra:** checar `find ios -name "*.mm" -path "*/ReactCodegen/*"` — se vazio, prebuild não terminou

**Problema 3:** Simulador mostra boilerplate "Open up App.tsx"
- **Causa:** Metro bundler não estava rodando junto com o simulador
- **Fix:** build com bundle embutido usando `-configuration Debug` + `BUNDLE_IDENTIFIER`
- **Ou:** rodar `npx expo start --ios` para desenvolvimento

**Problema 4:** Pod install falha (Headers Public vazio)
- **Causa:** prebuild incompleto deixou ios/ em estado inconsistente
- **Fix:** `cd ios && pod deintegrate && pod install --repo-update`

### Processo Correto (validado):
```bash
# 1. Prebuild limpo
cd /projeto && npx expo prebuild --clean --platform ios

# 2. Verificar ReactCodegen (OBRIGATÓRIO)
find ios -name "*.mm" -path "*/ReactCodegen/*" | head -3
# Se vazio: rodar prebuild novamente

# 3. Pod install
cd ios && pod install

# 4. Archive
xcodebuild archive \
  -workspace *.xcworkspace \
  -scheme SCHEME \
  -configuration Release \
  -archivePath /tmp/APP.xcarchive \
  CODE_SIGN_STYLE=Manual \
  DEVELOPMENT_TEAM=TY646U2BYX \
  CODE_SIGN_IDENTITY="Apple Distribution" \
  PROVISIONING_PROFILE_SPECIFIER="UUID_DO_PROFILE"

# 5. Export
xcodebuild -exportArchive \
  -archivePath /tmp/APP.xcarchive \
  -exportPath /tmp/APP-export \
  -exportOptionsPlist /tmp/ExportOptions.plist

# 6. Upload
xcrun altool --upload-app -f /tmp/APP-export/APP.ipa \
  -t ios -u EMAIL -p APP_SPECIFIC_PASSWORD
```
