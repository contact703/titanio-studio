### Pesquisa Anthropic Prompt Engineering (03/2025)
Técnicas principais documentadas:
1. Role+Context+Task+Format+Constraints — estrutura base
2. Be clear and direct — tratar Claude como novo funcionário brilhante
3. Add context/motivation — explicar o PORQUÊ das instruções
4. Few-shot com 3-5 exemplos em <examples> tags
5. XML tags para separar instruções/contexto/input
6. Long context: colocar dados no TOPO, query embaixo (até +30% qualidade)
7. Ground responses in quotes para documentos longos
