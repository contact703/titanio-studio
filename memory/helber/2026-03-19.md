### 6. Problemas identificados
- **Skills customizadas não carregam**: `openclaw skills list` só mostra bundled skills. Necessário investigar formato correto ou usar clawhub.
- **Mudança de modelo no TUI**: O TUI permite mudar modelo por sessão (override), mas o config.get mostra o modelo primário do gateway (anthropic/claude-sonnet-4-6). Eu não conseguia ver a mudança porque o TUI faz override local na sessão, não no config global.
- **RAM baixa**: 28k-44k pages free (threshold 50k) — monitorar

## Lições aprendidas
- O TUI do OpenClaw permite mudar modelo por sessão sem alterar o config global
- Quando Helber diz "mudei o modelo", ele pode ter mudado no TUI (override de sessão), não no openclaw.json
- Skills customizadas na pasta `~/.openclaw/workspace/skills/` NÃO são automaticamente carregadas pelo gateway
- Muitos modelos free do OpenRouter não suportam tool use (retornam 404)
- Puppeteer funciona para gerar PDFs no Mac mini
