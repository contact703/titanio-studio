# Gideão IA — Análise Competitiva

## O que é
Assistente visual que VÊ a tela do usuário em tempo real e guia passo a passo.
Foco: "pessoas que travam em cursos porque a tela é diferente"

## Stack deles
- Screen Capture: Extensão/app
- Vision: GPT-4 Vision (analisa screenshot, identifica software/menus/onde clicar)
- STT: Whisper ou similar
- TTS: ElevenLabs ou similar
- Backend: orquestra tudo

## Como poderíamos replicar/superar
- Screen Capture: Extensão Chrome
- Visão: GPT-4 Vision API (~$5-10/1000 usos)
- STT: Whisper local (grátis)
- TTS: Edge TTS (grátis) ou ElevenLabs
- Backend: Node.js ou FastAPI
- Custo estimado: $30-80/mês para ~1000 interações

## Diferencial deles
- Marketing focado em "pessoas que travam em cursos"
- Voz natural e conversacional
- Guia específico por tela
