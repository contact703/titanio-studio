## Configuração de Domínio Pendente
O Helber confirmou que têm o domínio `titanio.studio` no Namecheap.

**Para finalizar, precisa adicionar no Namecheap (DNS):**
```
CNAME kiteme → kiteme-website-production.up.railway.app
CNAME kidshq → [domínio do kidshq]
```

Depois adicionar custom domain no Railway em cada projeto.
