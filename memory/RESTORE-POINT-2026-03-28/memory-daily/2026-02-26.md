### Lições técnicas
- Elementor `_elementor_data` é uma string JSON dentro do meta field JSON (double-encoded)
- Regex simples pode falhar pois o dado interno usa escaped quotes
- Solução: parsear como JSON, modificar programaticamente, re-serializar
- Depois de editar _elementor_data via API, precisa limpar o cache do Elementor (Tools > Limpar)
- WordPress httpOnly cookies (wordpress_logged_in_*) precisam ser extraídos via CDP Network.getCookies
- Google Drive `/preview` retorna 200 mas bloqueia embed via CSP frame-ancestors
