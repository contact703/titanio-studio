**Rotina semanal (30 minutos toda segunda-feira):**
1. Rodar `scripts/gerar-relatorio.sh` — gera relatório atualizado + alerta de prazos
2. Verificar `processos-ativos.json` — atualizar status se houver mudanças
3. Checar este dashboard — confirmar próximas ações

3. Atualizar este dashboard

**Quando aprovado:**
1. Atualizar status no `processos-ativos.json`
2. Registrar valor aprovado neste dashboard
3. Criar pasta `/projetos/fundos/contratos/[nome-do-fundo]/`
4. Documentar próximos passos (assinatura de contrato, prestação de contas)
