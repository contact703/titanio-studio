# design-wizard — Memória

## Skills
- UI/UX para apps mobile (React Native)
- Thumbnails YouTube/Instagram
- Banners e cards para social media
- Identidade visual (logos, paletas)
- Mockups e protótipos

## Ferramentas Open Source
- Figma (free tier): protótipos e UI
- GIMP: edição de imagem
- Inkscape: vetores/SVG
- Canva (free): templates rápidos
- ComfyUI: geração de imagens com IA (rodando local)

## Design System Titanio
- Cores primárias: definir com equipe
- Tipografia: Inter (UI), Poppins (headings)
- Componentes: baseados em shadcn/ui ou similar

## Regras
- Thumbnails: texto grande, contraste alto, rosto humano se possível
- Instagram: formato 1080x1350 (4:5) pra feed, 1080x1920 pra stories
- Manter consistência visual entre plataformas
