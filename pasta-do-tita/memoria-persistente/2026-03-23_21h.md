## 🎓 Lições Aprendidas (Última Sessão)

### 1. **Backend é crítico — precisa de watchdog agressivo**
- Criash sem motivo aparente
- Solução: Monitor a cada 5 minutos, kill + restart se não responder
- **Aplicar:** Todos os 3 Macs precisam de watchdog (não apenas Eduardo)

### 2. **Sincronização = é SUA responsabilidade avisar**
- Eduardo perguntou "ta sincronizando?"
- Resposta automática: cada Mac é independente, Gateway é o nexo
- **Documentar:** Como fazer sincronização bidirecional (se necessário)

### 3. **Paths hardcoded são problema perene**
- Helber e Tiago têm Macs novos, paths diferentes
- **Padrão criado:** Script substitute paths automaticamente
- **Usar sempre:** Detectar path real, não assumir

### 4. **API key SQLite > browser login**
- Browser login não funciona em servidor headless
- SQLite insert é 100% automático, zero falhas
- **Padrão:** N8n sempre via API + SQLite

### 5. **Documentação em "partes" é melhor que um mega-doc**
- 1 arquivo 50KB = ninguém lê
- 10 arquivos 5KB cada = tudo lido
- **SETUP-DEFINITIVO-TIAGO:** 10 partes = 100% claro
