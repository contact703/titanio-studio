## 💡 Lições Aprendidas

- **PIX/Stripe no iOS = REJEITADO** — Apple exige IAP StoreKit exclusivamente para pagamentos in-app
- **expo prebuild --clean DEVE completar** antes do xcodebuild (caso contrário: ReactCodegen .mm not found)
- **EAS + react-native RC = incompatível** — sempre build local
- **subagent.steer()** funciona para adicionar instruções a subagentes já rodando
- **App-specific passwords** só para tools/apps terceiros, NÃO para login browser em ASC
