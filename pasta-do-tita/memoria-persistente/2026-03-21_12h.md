## 🎓 Lições Aprendidas / Reforçadas

1. **Documentação clara > Acessibilidade rápida**
   - Deployment package com 3 arquivos > instruções vagas
   - Cada colega pode instalar autonomamente sem pedir ajuda

2. **Shared vs Local é crítico**
   - Especialistas compartilhados (30) não devem ser duplicados
   - Credenciais pessoais (GitHub token) devem ser geradas por cada um
   - .env ignorado no git previne vazamentos

3. **Scripts automáticos salvam tempo**
   - install-dashboard.sh em 5 min faz o que levaria 30 min manual
   - Reduz chance de erro (configuração, missing steps)

4. **Sincronização via git funciona bem**
   - memoria-especialistas sincroniza automático
   - Qualquer Mac pode puxar updates dos colegas
   - Versioning natural (git history)
