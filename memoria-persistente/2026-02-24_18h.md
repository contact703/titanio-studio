## Lições Aprendidas
1. **RAID degraded = dados inacessíveis**: Quando AppleRAID está degraded com 0% rebuild, macOS não monta o volume. Precisa resolver o RAID primeiro.
2. **Rate limit cascata**: Quando Anthropic retorna 429, o OpenClaw tenta múltiplas vezes mas não faz fallback para outro provider automaticamente (o grupo Gospia ficou sem resposta)
3. **Flush precisa de fallback**: Sempre ter um local alternativo para salvar dados quando o volume principal está offline
