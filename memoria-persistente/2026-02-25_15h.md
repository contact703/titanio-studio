## Lições Aprendidas
1. **Browser automation para Elementor é frágil**: timeouts frequentes, botões disabled, estado inconsistente. Para edições complexas, é melhor documentar os passos e pedir ao humano.
2. **Apple RAID impede mount direto**: mesmo após fsck bem-sucedido, partições Apple_RAID não montam com `mount -t hfs`. Precisa de abordagem de clone/imagem.
3. **fsck pode "reparar" mas não significa que vai montar**: disk6s2 passou no fsck mas não monta.
4. **Sempre clonar antes de reparar**: nunca tentar mais reparos no disco original sem ter backup bit-a-bit.
5. **Sub-agentes com browser são propensos a timeout**: LLM requests timeando ao usar browser automation pesada. Considerar tarefas menores ou abordagem sequencial.
