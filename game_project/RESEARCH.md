# Pesquisa: Jogos Hipercasuais 2024-2025

## 📊 O que são Jogos Hipercasuais?

Jogos com mecânicas extremamente simples, sessões curtas (30s-2min), e monetização por anúncios.

---

## 🎮 Mecânicas que Mais Vendem

### Top Performers:
1. **Number/Evolution Runners** (Count Masters, Tall Man Run)
   - Coletar números/multiplicadores
   - Evoluir personagem durante corrida
   - Gates de multiplicação

2. **Satisfying/ASMR Games** (Perfect Slices, ASMR Slicing)
   - Feedback tátil satisfatório
   - Sons agradáveis
   - Progressão visual clara

3. **Stacking/Timing** (Stack, Helix Jump)
   - Um toque para ação
   - Timing preciso = recompensa

4. **Idle + Clicker Elements**
   - Progresso mesmo offline
   - Upgrades simples

5. **Physics-Based** (Bridge Race, Hair Challenge)
   - Física simples mas divertida
   - Coleta de recursos

---

## 🎨 Estética que Funciona

### Cores:
- **Paletas pastéis** - suaves aos olhos
- **Cores vibrantes** - destaque para elementos importantes
- **Gradientes** - modernidade
- **Contraste alto** - legibilidade

### Estilo Visual:
- **Low Poly** - performance + charme
- **Flat Design** - limpo e moderno
- **Cartoon estilizado** - apelo universal
- **Personagens simples** - fácil identificação

### Partículas e Feedback:
- Confetti em vitórias
- Shake de câmera sutil
- Trails de movimento
- Pop de UI elements

---

## 👤 Personagens que Engajam

### Características:
- **Proporções exageradas** (cabeça grande)
- **Expressões simples** mas claras
- **Customização** (skins, cores)
- **Silhueta reconhecível**

### Temas Populares:
- Stickman/humanoid genérico
- Animais fofinhos
- Objetos antropomorfizados
- Blobs/slimes coloridos

---

## 💰 Monetização

### Modelo Padrão:
1. **Interstitial Ads** - entre níveis (a cada 2-3 níveis)
2. **Rewarded Video** - revive, 2x moedas, skip level
3. **Banner** - discreto no gameplay
4. **IAP leve** - remover ads, skins premium

### Métricas Alvo:
- **CPI**: < $0.30
- **D1 Retention**: > 40%
- **D7 Retention**: > 15%
- **Playtime/day**: 15-30 min

---

## 🛠️ Desenvolvimento Unity

### Estrutura Recomendada:
```
/Assets
  /Scripts
    GameManager.cs
    PlayerController.cs
    LevelManager.cs
    UIManager.cs
    AdManager.cs
  /Prefabs
  /Materials
  /Models
  /Scenes
  /Audio
```

### Boas Práticas:
1. **Object Pooling** - performance
2. **Scriptable Objects** - configuração de níveis
3. **Event System** - desacoplamento
4. **DOTween** - animações suaves
5. **TextMeshPro** - UI text

### Plugins Essenciais:
- Unity Ads / AdMob
- Analytics (Firebase/GameAnalytics)
- DOTween (animações)
- Easy Save (progresso)

## 📈 Tendências 2024-2025

1. **Hybrid Casual** - hipercasual + meta progression
2. **Satisfying mechanics** - ASMR continua forte
3. **AI-generated content** - níveis procedurais
4. **Social features** - leaderboards, challenges
5. **Narrative light** - história simples mas presente

---

## 🎯 Fórmula de Sucesso

1. **Core Loop em 3 segundos** - entendimento instantâneo
2. **Fail fast, retry fast** - sem frustração
3. **Always winning feeling** - progresso constante
4. **One more level** - vício saudável
5. **Share moments** - viral potential
